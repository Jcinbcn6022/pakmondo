# PakMondo

Field-journal-styled adventure travel packing app. React + Vite + Supabase.

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Building for production

```bash
npm run build
```

Output goes to `dist/`.

## Deploying

This app deploys cleanly to Vercel, Netlify, Cloudflare Pages, or any static host.

For Vercel: connect this repo, framework auto-detects as Vite, click Deploy.
